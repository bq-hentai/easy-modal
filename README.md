easy-modal
===========

** No *md Wrap **

## Use

```
new Modal(content, opts);
```

## APIS

### create

```
var modal = new Modal(content, opts);
```

#### opts

```
var opts = {
  title: String,
  ok: fn | null | undefined | false,
  ccl: fn | null | undefined | false
}
```

### close

```
modal.close();
```
